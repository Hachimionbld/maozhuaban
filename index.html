校园十大优秀青年评比

基于哈希表与二叉排序树数据结构，实现学生提名、投票、查询与排行榜功能。每一位在校学生都可以为自己认为优秀的同学投出宝贵的一票！

📋 哈希表存储 · 除留余数法
⭐ 二叉排序树 · 排行榜

🗳️ 功能一：提名优秀学生和投票

输入学生姓名、专业和年级为其提名或投票。已提名学生将自动累加票数。

🔍 功能二：查看提名学生基本信息

输入学生姓名、专业和年级，查询该同学的提名信息与当前票数。

📊 功能三：各提名学生票数总览

显示哈希表中所有提名学生的专业、年级、姓名、票数及查找长度。

序号 姓名 专业 年级 票数 关键字 (k) 哈希地址 查找长度 (si)

📈 平均查找长度 (ASL)

0 = 总查找长度 / 学生数（0）

#️⃣ 哈希表存储结构可视化

表长 20，哈希函数 hash(key)=key%19，冲突时 adr=(adr+key)%19

🏅 功能四：十大优秀青年排行榜

基于二叉排序树（BST），按票数从高到低显示前 10 名同学。

🔄 功能五：退出系统 / 重置数据

清空所有提名与投票数据，恢复为初始测试数据状态。
